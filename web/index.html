How It Works

1

Add the Bot

Invite Zentility to your Discord server

2

Link Account

Use /linkroblox to authorize your Roblox account with OAuth 2.0

3

Set Up Monitoring

Use /monitor to choose which users to track and where to send notifications

4

Get Notified

Receive instant Discord notifications when users start playing Roblox

Available Commands

/linkroblox

Connect your Roblox account using OAuth 2.0

/monitor @user #channel [role]

Set up presence monitoring for this server

/status

View bot status and monitoring information

/testembed

Send a preview notification embed

/unlink

Revoke bot access to your Roblox account
